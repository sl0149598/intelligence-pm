Import Emails

Add emails from Outlook. New emails merge with existing data — no duplicates.

📄
Drop Outlook CSV here or click to browse
File → Open & Export → Import/Export → CSV
✉️
Drag an email from Outlook to your Desktop first
It saves as a .eml or .msg file. Then drag that file here.
.eml files are fully supported. .msg files are read as plain text (some formatting may be lost).
.pst files can't be read directly in a browser
The quickest path is to export your emails to CSV in Outlook, then use the CSV tab above.
1In Outlook, go to File → Open & Export → Import/Export
2Choose Export to a File → Comma Separated Values (.csv)
3Select the folder or mailbox you want (e.g. Cannon Beach inbox)
4Save the .csv, then switch to the CSV Export tab to import it
Single emails: Drag any email from Outlook to your Desktop (saves as .eml), then use the Drag & Drop tab.
